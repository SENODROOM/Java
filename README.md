## üöÄ Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- A text editor or IDE (IntelliJ IDEA, Eclipse, VS Code, etc.)

### Running the Examples

1. Clone the repository:
```bash
git clone https://github.com/SENODROOM/Java.git
cd Java
```

2. Navigate to the desired topic folder:
```bash
cd "Conditional Statements"
# or
cd "Input and Output"
```

3. Compile and run any Java file:
```bash
javac YourFileName.java
java YourFileName
```

### How to Contribute

1. **Fork the repository** - Click the 'Fork' button at the top right of this page
2. **Create your feature branch**:
   ```bash
   git checkout -b feature/AmazingExample
   ```
3. **Add your changes** - Add new examples, fix bugs, or improve existing code
4. **Commit your changes**:
   ```bash
   git commit -m 'Add some AmazingExample'
   ```
5. **Push to your branch**:
   ```bash
   git push origin feature/AmazingExample
   ```
6. **Open a Pull Request** - Go to the original repository and click 'New Pull Request'

### Contribution Guidelines

- Write clean, well-commented code
- Follow Java naming conventions

- Test your code before submitting
